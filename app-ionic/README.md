# Aplicativo - Ionic 5 + Angular 11

## Requisitos
* Node v14.17.0
* NPM
* Ionic v6.16.1

## Instalando
```bash
npm i
```

## Iniciando aplicativo modo web
```bash
ionic serve
```

Para acessar o aplicativo é necessário o container do backend estar executando

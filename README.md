# Aplicação exemplo

#### Backend - Node.js + Express + Mongo 
#### Frontend - Angular 12
#### Aplicativo - Ionic 5 + Angular 11
#### Docker + Docker Compose

## Requisitos
* Node v14.17.0
* NPM
* Mongo
* Angular CLI 12.0.1
* Ionic v6.16.1
* Docker

## Rodando via docker
```bash
docker-compose up -d
```

Para acessar a aplicação ao termino da execução do docker compose -> [http://localhost](http://localhost)
